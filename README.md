# 🛒 Retail Store Performance Dashboard

## 📌 Project Overview

The **Retail Store Performance Dashboard** is an end-to-end Data Analytics project designed to analyze retail business performance using **SQL Server, Microsoft Excel, and Tableau**.

The objective of this project is to transform raw retail data into meaningful business insights by analyzing:

- Sales performance
- Revenue trends
- Store performance
- Product analysis
- Customer purchasing behavior
- Regional sales patterns

This project follows a complete analytics workflow:

**Data Modeling → SQL Analysis → Excel Dashboard → Tableau Visualization → Business Insights**

# 🗄️ Database Structure

The project database contains the following tables:

### 📍 Stores
Contains store-level information.

### 📦 Products
Contains product details and categories.

### 👥 Customers
Contains customer information.

### 🧾 Sales
Stores transaction-level sales records.

### 📋 SalesDetails
Contains product-level transaction details.
